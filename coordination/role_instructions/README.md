## üöÄ Getting Started

### 1. Initialize Framework
```bash
# Deploy complete MDD framework with all roles
npx ruv-swarm mdd-deploy --project "hnp-modernization" --framework "symphony"
```

### 2. Configure HNP Context
```bash
# Load project-specific context and environment
source /home/ubuntu/cc/hedgehog-netbox-plugin/.env
npx ruv-swarm context-load --project-path "/home/ubuntu/cc/hedgehog-netbox-plugin"
```

### 3. Execute MDD Symphony
```bash
# Start coordinated MDD pipeline execution
npx ruv-swarm task-orchestrate --task "HNP MDD Modernization" --strategy "mdd-pipeline"
```

### 4. Monitor Progress
```bash
# Real-time coordination monitoring
npx ruv-swarm swarm-monitor --framework "mdd-symphony" --real-time true
```
