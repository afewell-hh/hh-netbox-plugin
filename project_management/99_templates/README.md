# Project Templates Directory

**Purpose**: Reusable templates to ensure consistency across all project documentation and processes.

## Available Templates

### Planning Templates
- **sprint_planning_template.md** - Sprint planning document structure
- **risk_assessment_template.md** - Risk identification and mitigation

### Execution Templates  
- **task_definition_template.md** - Task specification format
- **status_report_template.md** - Regular status update format

### Coordination Templates
- **agent_instruction_template.md** - Agent role and task instructions
- **retrospective_template.md** - Sprint retrospective format

### Documentation Templates
- **project_documentation_standards.md** - Documentation guidelines

## Usage Guidelines

1. **Copy, Don't Modify** - Always copy templates to working directories
2. **Maintain Consistency** - Use templates for all similar documents
3. **Suggest Improvements** - Templates evolve based on usage
4. **Version Control** - Templates are versioned with the project

## Template Standards

- Clear section headers
- Placeholder text in [BRACKETS]
- Examples where helpful
- Concise instructions
- Consistent formatting

## Creating New Templates

1. Identify repeated document patterns
2. Extract common structure
3. Add clear placeholders
4. Include usage instructions
5. Test with real use case
