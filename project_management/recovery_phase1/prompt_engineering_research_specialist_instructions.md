## Mission Statement

**Primary Objective**: Research and document the latest, most authoritative best practices for creating agent instructions and prompt engineering specifically for Claude-based agents, with focus on applications to complex, long-running software development projects.

**Strategic Importance**: This research directly impacts the success of the entire project recovery effort. All subsequent agent instructions, onboarding materials, and project management processes will be based on your findings.

## Specific Research Questions to Address

### Critical Questions for HNP Application

1. **Context Management**: What are the most effective strategies for managing agent context in long-running software projects?

2. **Hierarchical Delegation**: How should manager agents structure instructions for task agents to maximize effectiveness?

3. **Process Compliance**: What techniques are most effective for ensuring agents consistently follow established processes (git commits, documentation updates)?

4. **Error Prevention**: How can agent instructions be designed to prevent destructive actions when agents encounter unexpected situations?

5. **Role Specialization**: What are the optimal instruction patterns for different agent roles (orchestrator, manager, specialist)?

6. **Quality Assurance**: How can quality validation be built into agent instructions without overwhelming the agent's context?

7. **Escalation Protocols**: What are the best practices for designing clear escalation and communication paths in agent hierarchies?

8. **Knowledge Transfer**: How can agent instructions be designed to ensure effective knowledge transfer between agents working on related tasks?

9. **Environment Setup Efficiency**: How can agent instructions eliminate repeated environment discovery cycles (NetBox Docker, HCKC, ArgoCD clusters) that waste significant agent time and context?

### Implementation-Focused Questions

10. **Onboarding Optimization**: What are the most effective approaches for modular, role-based agent onboarding?

11. **Documentation Integration**: How should agent instructions integrate with project documentation and knowledge management?

12. **Performance Measurement**: What metrics and evaluation criteria are most effective for assessing agent instruction quality?

13. **Iterative Improvement**: How can agent instruction effectiveness be continuously improved based on real-world performance?

14. **Environment Knowledge Embedding**: What are the most effective techniques for embedding operational environment knowledge in agent instructions to prevent repeated discovery cycles?

15. **Operational Standardization**: How can complex development environment operations be standardized and embedded in agent training to maximize efficiency?
