# Project Execution Directory

**Purpose**: Active work tracking, sprint execution logs, and quality assurance for the HNP project.

## Subdirectories

### sprint_logs/
- **current/** - Active sprint execution logs
- **archive/** - Historical sprint logs (moved after sprint completion)

### task_tracking/
- **active_tasks.md** - Currently active tasks with status
- **task_registry.md** - Complete task inventory with history

### quality_assurance/
- **test_results/** - Test execution reports and results
- **qa_checklists.md** - Quality validation checklists

## Key Documents

- **delivery_tracking.md** - Milestone delivery status and timelines

## Execution Process

1. **Task Creation** - New tasks logged in task_tracking/
2. **Sprint Execution** - Daily updates to sprint_logs/current/
3. **Quality Validation** - Test results and QA checklists maintained
4. **Delivery Tracking** - Milestone progress updated regularly

## Update Frequency

- **Daily**: Sprint logs, task status
- **Per Task**: Test results as completed
- **Weekly**: Delivery tracking updates

## Archive Process

- Sprint logs moved to archive/ at sprint completion
- Completed tasks remain in registry for reference
- Test results preserved for compliance
