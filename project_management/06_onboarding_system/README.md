## üéÆ Usage Patterns

### For Human Managers Creating Agents

1. **Assess Task Complexity**:
   - Simple task ‚Üí Specialist agent
   - Feature delivery ‚Üí Manager agent  
   - Multi-feature coordination ‚Üí Orchestrator agent

2. **Select Onboarding Path**:
   - All agents: Universal Foundation (required)
   - Role-specific: Choose appropriate track
   - Advanced: Environment Mastery if needed
   - All agents: Escalation Protocols (required)

3. **Use Templates**:
   - Location: `99_templates/AGENT_INSTRUCTION_TEMPLATES.md`
   - Research-validated templates for each role
   - Customize for specific tasks
   - Include environment and process context

4. **Monitor and Support**:
   - Respond to escalations rapidly (<30 minutes)
   - Provide learning-oriented guidance
   - Update processes based on escalation patterns

### For Agents Using Onboarding System

1. **Complete Universal Foundation**:
   - Environment mastery (eliminate discovery cycles)
   - Process compliance (git, testing, documentation)
   - Escalation culture (when uncertain, ask)

2. **Master Role-Specific Skills**:
   - Deep expertise in assigned domain
   - Coordination patterns appropriate to role
   - Quality standards for deliverables

3. **Apply Escalation Protocols**:
   - Escalate uncertainty immediately
   - Provide sufficient context for rapid resolution
   - Learn from escalation guidance

4. **Contribute to Improvement**:
   - Identify process gaps through experience
   - Share effective patterns with team
   - Support continuous system improvement
