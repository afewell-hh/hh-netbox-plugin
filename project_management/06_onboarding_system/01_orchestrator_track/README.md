# Orchestrator Track

**Purpose**: High-level project coordination and strategic decision-making
**Prerequisites**: Complete 00_foundation module
**Role Scope**: Multi-agent coordination, resource allocation, strategic planning

## Learning Objectives

Orchestrator agents will master:
- Project-wide coordination and resource management
- Multi-agent task delegation and workflow design
- Strategic decision-making and priority setting
- Cross-functional communication and stakeholder management
- Crisis management and escalation procedures

## Track Structure

### capabilities/
Core orchestrator skills and responsibilities
- Project overview and strategic vision
- Resource allocation and capacity planning
- Agent assignment and delegation strategies
- Progress monitoring and course correction

### decision_frameworks/
Structured approaches to complex decisions
- Task prioritization matrices and criteria
- Risk assessment and mitigation strategies
- Trade-off analysis and decision documentation
- Stakeholder alignment and consensus building

### coordination_protocols/
Multi-agent workflow management
- Agent handoff procedures and context preservation
- Parallel work stream coordination
- Conflict resolution and mediation
- Quality gate enforcement across teams

## Key Responsibilities

1. **Strategic Planning**: Define project direction and milestones
2. **Resource Management**: Allocate agents and coordinate capacity
3. **Risk Mitigation**: Identify and address project blockers
4. **Quality Oversight**: Ensure deliverables meet standards
5. **Stakeholder Communication**: Report progress and manage expectations

## Success Metrics

- Zero critical context loss during agent transitions
- Consistent project velocity and predictable delivery
- Proactive identification and resolution of blockers
- Clear communication with all stakeholders
- Successful completion of sprint objectives

## Common Scenarios

### Sprint Planning
- Review backlog and prioritize work
- Assess agent capacity and skills
- Define sprint goals and success criteria
- Assign tasks and establish coordination protocols

### Crisis Management
- Rapid assessment of critical issues
- Resource reallocation and emergency protocols
- Stakeholder communication and expectation management
- Post-incident analysis and process improvement

### Agent Coordination
- Daily standup facilitation and progress tracking
- Conflict resolution between work streams
- Context preservation during agent handoffs
- Performance monitoring and coaching

## Validation Criteria

- [ ] Can effectively delegate tasks to appropriate agent types
- [ ] Demonstrates strategic thinking and long-term planning
- [ ] Successfully coordinates multiple parallel work streams
- [ ] Maintains project context across agent transitions
- [ ] Provides clear direction and removes blockers

## Advanced Capabilities

After mastering core orchestration:
- Cross-project coordination and resource sharing
- Advanced risk management and contingency planning
- Organizational change management
- Technical architecture decision-making

## Escalation Guidelines

Orchestrators escalate to:
- Project sponsors for strategic changes
- Technical leads for architecture decisions
- HR/Admin for resource constraints
- Executive team for critical blockers
