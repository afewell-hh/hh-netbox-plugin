# Current Project State

**Purpose**: Always-current snapshot of the HNP project status. This directory contains the most up-to-date information about the project's current state.

## Key Documents

- **project_overview.md** - Executive summary including:
  - Current project phase and status
  - Key achievements and milestones
  - Active issues and blockers
  - Overall project health

- **active_sprints.md** - Current sprint information:
  - Sprint goals and objectives
  - Task progress and completion status
  - Sprint burndown metrics
  - Daily standup summaries

- **resource_allocation.md** - Current team assignments:
  - Active agent roles and responsibilities
  - Task assignments by agent
  - Workload distribution
  - Availability and capacity

- **immediate_priorities.md** - Next actions required:
  - Critical path items
  - Blocking issues requiring resolution
  - Upcoming deadlines
  - Decision points

## Update Frequency

- **Daily**: Sprint progress, task status, immediate priorities
- **Sprint Boundaries**: Resource allocation, sprint documentation
- **As Needed**: Project overview when major changes occur

## Usage Guidelines

1. **Always check here first** when starting work on the project
2. **Update immediately** when completing tasks or encountering blockers
3. **Keep concise** - detailed information belongs in other directories
4. **Use templates** from `99_templates/` for consistency

## Integration Points

- Links to active work in `02_execution/`
- References current plans from `01_planning/`
- Escalations tracked in `03_coordination/`
