## Project Overview

### Mission Statement
Create a comprehensive, user-friendly solution that enables non-Kubernetes experts to deploy and manage the external Kubernetes infrastructure required to support their Hedgehog ONF fabric deployments.

### Strategic Context
- **Parent Project**: Hedgehog NetBox Plugin (HNP) - MVP2 Complete
- **Problem**: HNP requires external GitOps and monitoring tools, but customers lack Kubernetes expertise
- **Solution**: Opinionated, automated platform for deploying Hedgehog External Management Kubernetes (HEMK)
- **Target Users**: Traditional enterprise network engineers new to Kubernetes
